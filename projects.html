User Mangement Web Application

"FLIES" AKA "Fast, Lightweight, Interactive, Efficient, Secure" is a full-stack web application built with React (Material UI) for the frontend and Go (Gin) for the backend. It provides authentication, role-based access control, and user management capabilities.

https://github.com/shiatomaro/FLIES
AI Discord bot: Go Ask Away

This is a Discord bot built with Go, leveraging the DiscordGo library to interact with Discord and the Gemini API to provide AI-powered responses to user queries. The bot registers a slash command /ask which allows users to ask any question and receive AI-generated responses.

https://github.com/shiatomaro/go-gemini-discord-bot
Web Scraper with Colly (Go)

This is a simple web scraper built with Go using the Colly web scraping framework. It extracts product details (name, description, price, and image URL) from webscraper.io and saves the data as a JSON file.

Live Demo https://github.com/shiatomaro/go-web-scraper-using-colly
